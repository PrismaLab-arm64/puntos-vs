# PUNTOS VS
### Marcador Universal Multijugador y Tablero Digital

PUNTOS VS es una aplicacion web progresiva (PWA) de alto rendimiento diseñada para registrar puntuaciones en tiempo real. Su interfaz de alto contraste y bajo consumo es ideal para torneos, bares, clubes sociales y competencias domesticas.

[ ACCESO DIRECTO A LA APLICACION ]
>>> https://prismalab-arm64.github.io/puntos-vs/ <<<
(Compatible con Android, iOS, Windows y MacOS - No requiere descarga)

CARACTERISTICAS TECNICAS

+ ARQUITECTURA FLEXIBLE
  - 4 Equipos (A, B, C, D) con colores neon distintivos.
  - Soporte para multiples jugadores por faccion.
  - Modos: Individual (FFA) o Cooperativo (Team Deathmatch).

+ HERRAMIENTAS INTEGRADAS
  - Calculadora Aritmetica: Suma combinaciones complejas sin salir de la app.
  - Historial de deshacer (Undo): Correccion inmediata de errores de digitacion.
  - Wake Lock API: Mantiene la pantalla encendida durante la sesion.

+ ESPECIFICACIONES
  - Stack: HTML5, CSS3, Vanilla JS.
  - Diseño: Dark Mode / Neon UI / Responsive.
  - Despliegue: GitHub Pages (SSL Seguro).
